# CODION

> 날씨와 일정, 보유 의류 정보를 기반으로  
> 오늘 입기 좋은 옷을 추천해주는 서비스

---

## 📌 프로젝트 소개

CODION은  
사용자의 **날씨 환경, 활동 유형, 보유 의류**를 종합하여  
**“오늘 입을 옷을 고민하는 비용”을 줄이기 위해** 시작된 프로젝트입니다.

- 단순 추천이 아닌 **선택을 돕는 체크리스트 기반 추천**
- 사용자 행동 데이터를 활용한 **추천 로직 개선 구조**
- 서비스 확장을 고려한 **관리자 대시보드 제공**

---

## 🎯 핵심 목표

- 날씨·환경·활동 맥락을 반영한 현실적인 옷 추천
- 추천 결과에 대한 사용자 피드백 수집 및 개선
- 데이터 기반 의사결정이 가능한 관리자 도구 제공

---

## 🧩 주요 기능

### 사용자 기능
- 보유 의류 등록 / 수정 / 삭제
- 날씨 기반 코디 추천
- 추천 결과 체크리스트 제공
- 추천 이력 확인

### 관리자 기능
- 대시보드 Overview
- 사용자 / 추천 / 클릭 로그 통계
- 추천 전략별 성과 확인

---
